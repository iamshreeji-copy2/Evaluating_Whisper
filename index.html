About

This project explores sub-10M parameter neural vocoders derived from BigVGAN for low-resource Indic languages. We evaluate subjective (MOS) and objective metrics (PESQ, STOI, SNR, MCD, MSD, PCC, FAD, NISQA) and publish code & samples for transparent comparison.
